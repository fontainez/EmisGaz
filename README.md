# EmisGaz - Emissions Analysis Project

## Overview
EmisGaz is a comprehensive Python-based data analysis project focused on emissions data processing, validation, and exploratory analysis. This repository contains a series of Python scripts that perform end-to-end data processing from raw data loading to advanced analytics and visualization.

## Project Structure

### Core Scripts
- `01_data_loading.py` - Initial data loading and basic preprocessing
- `02_data_cleaning.py` - Data cleaning and quality checks
- `03_dataset_analysis.py` - Comprehensive dataset analysis
- `04_data_preparation.py` - Data preparation for modeling
- `05_data_validation.py` - Data validation and integrity checks
- `06_data_cleaning_enhancement.py` - Enhanced data cleaning procedures
- `07_validate_enhanced_data.py` - Validation of enhanced data
- `08_exploratory_data_analysis.py` - Exploratory data analysis and visualization

### Results
The `results/` directory contains generated visualizations including:
- Climate trends analysis
- Correlation matrices
- Emissions vs temperature scatter plots
- GHG composition charts
- Seasonal patterns
- Sector contributions analysis

## Installation

1. Clone this repository:
```bash
git clone https://github.com/fontainez/EmisGaz.git
cd EmisGaz
```

2. Install required dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the scripts in numerical order for complete data processing pipeline:

```bash
python 01_data_loading.py
python 02_data_cleaning.py
# ... continue through script 08
```
